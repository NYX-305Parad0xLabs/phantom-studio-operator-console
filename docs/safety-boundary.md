# Safety Boundary

- **Original or licensed content only**: Operators may only submit media sources for which the studio holds rights or which are original synthetic creations with explicit disclosure.
- **No impersonation**: The console never facilitates work on real-person impersonations, stealth deepfakes, or politically deceptive content.
- **Human review first**: Even after rendering/exporting, everything stays gated behind the control plane’s human review decision before any publishing action.
- **No blind publishing**: Publishing is triggered explicitly from the console via the control plane; the UI only prepares export metadata.
- **Policy hooks**: The console surface enforces display-only hints (e.g., disclosure labels, protected identity flags) so operators can cross-check before advancing.
